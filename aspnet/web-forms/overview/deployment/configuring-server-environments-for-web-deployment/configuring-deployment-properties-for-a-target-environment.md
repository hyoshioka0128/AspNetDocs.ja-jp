## <a name="process-overview"></a>プロセスの概要

連絡先マネージャー ソリューション ビルドしてデプロイに使用するプロジェクト ファイルは、2 つの物理ファイルに分かれています。

- 特定の環境を含む 1 つのビルド設定 (*Env Dev.proj*、 *Env Stage.proj*など)。
