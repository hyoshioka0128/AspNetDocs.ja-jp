背景色のプロパティである、1 行には、"bc"を入力し、下向きの矢印を 1 回押します。

[背景色] のようなハイフンで区切られたプロパティに各単語の最初の文字を入力すると、IntelliSense には、一致するプロパティのみを表示するためのリストが絞り込ま。
