- 制限は、データを選択して処理する必要があるデータのサイズが実際に分析する必要があるデータだけを格納します。 たとえば、ユーザー プロファイルのデータ ストアから誕生年のみを選択するために、ユーザーが生まれた年ベースの構成を知りたいです。
