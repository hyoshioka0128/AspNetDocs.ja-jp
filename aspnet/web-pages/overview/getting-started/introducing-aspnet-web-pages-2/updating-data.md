> 特定の種類のデータベース操作を実行するその他のいくつかの方法はあります。 データベースのメソッドの一覧を見つけることができます、 [ASP.NET Web Pages API のクイック リファレンス](../../api-reference/asp-net-web-pages-api-reference.md#Data)します。
